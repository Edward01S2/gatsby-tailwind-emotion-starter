# Gatsby Tailwind Emotion Starter

## Usage

### Develop

```
npm run develop
```

### Build

```
npm run build
```
Your built file will be in `/public`

This project was based on [gatsby-plugin-tailwindcss/](https://github.com/muhajirframe/gatsby-plugin-tailwindcss/)
